### 错题 1／13｜た形变形：行く、来る、遊ぶ 三个全错了

> **出处：** 六-4、六-5、六-7
> **危险等级：** 🔴
> **标签：** #变形错误
> **语法点：** [G015] た形

#### 错在哪

六-4　行きます → ~~いいた~~ → **行った（いった）**
六-5　来ます → ~~くった~~ → **来た（きた）**
六-7　遊びます → ~~遊ぶんだ~~ → **遊んだ（あそんだ）**

#### 到底怎么回事

六-4：「行く」是た形/て形里唯一的特例。一般「く」结尾走「いて/いた」，但行く不走这条路——行く → 行って → 行った。你写的「いいた」说明还在套一般规则。

六-5：「来る」是V3不规则动词，只有两个V3：する和来る。来る的た形只能死记：来た（きた）。你写的「くった」看起来是把它当V1处理了。

六-7：遊ぶ属于「ぶ/む/ぬ」结尾，走拨音便：ぶ→んで→んだ。你写的「遊ぶんだ」保留了原形的「ぶ」，问题在于变形时原来的尾音要去掉，不是加在后面。

#### 一条规则解决

た形不需要单独记。先想て形，然后把「て」换成「た」，「で」换成「だ」。

```
た形 = て形换 て→た / で→だ

V1 く→いた（書いた）、ぐ→いだ（泳いだ）★行く→行った 特例
V1 ぶ/む/ぬ→んだ（遊んだ、飲んだ、死んだ）
V1 つ/う/る→った（待った、買った、帰った）
V1 す→した（話した）
V2 去る＋た（食べた、見た）
V3 した、来た（きた）
```

#### 对比

| 你写的 | 正确的 | 哪里出了问题 |
|--------|--------|------------|
| いいた | 行った | 行く是特例，不走一般く→いた |
| くった | 来た（きた）| 来る是V3，不能按V1变 |
| 遊ぶんだ | 遊んだ | 拨音便要去掉ぶ，不是保留 |

#### 练

- [ ] 1. 飲みます → ＿＿＿＿
- [ ] 2. 行きます → ＿＿＿＿（再写一遍）
- [ ] 3. 来ます → ＿＿＿＿
- [ ] 4. 死にます → ＿＿＿＿
- [ ] 5. 持ちます → ＿＿＿＿

### 错题 2／13｜また、もう、も——中文都能翻成"再"，日语完全不同

> **出处：** 四-10、八-3、八-6
> **危险等级：** 🔴
> **标签：** #词汇混淆
> **语法点：** [G004] もう/まだ/また/も

#### 错在哪

四-10　お茶を（　）一杯いかがですか。　~~また~~ → **もう**
八-3　明日（　）来てください。　~~も~~ → **また**
八-6　楽しかったです。（　）行きたいです。　~~もう~~ → **また**

#### 到底怎么回事

三道题三个词全搞混了。问题出在中文的"再"：

- "再来一杯"——这里的"再"是**追加数量**。日语用 **もう＋数量词**。你选了また，但また是"下次再"。
- "明天再来"——这里的"再"是**下次再做一遍**。日语用 **また**。你选了も，但も是"也"。
- "还想再去"——同样是"下次再"的意思。日语用 **また**。你选了もう，但もう单独用时是"已经"。

判断方法就一条：后面有没有跟数量词。

| 场景 | 选谁 | 例子 |
|------|------|------|
| 后面有数量词（一杯、一度、少し） | **もう** | もう一杯 |
| "下次再"、"又" | **また** | また来てください |
| "也" | **も** | 私も行きます |
| "已经"（单独用） | **もう** | もう食べました |

もう管"追加数量"和"已经"。また管"下次再来"。这是它们最核心的区别。

#### 对比

| 中文 | 你选的 | 正确的 |
|------|--------|--------|
| 再来一杯 | また | **もう**一杯（后面跟了数量词） |
| 明天再来 | も | **また**（下次再做） |
| 还想再去 | もう | **また**（同上） |

#### 练

- [ ] 1. ビールを（　）一本ください。
- [ ] 2. 来週（　）会いましょう。
- [ ] 3. 田中さん（　）来ました。
- [ ] 4. （　）少し待ってください。
- [ ] 5. 面白かったです。（　）見たいです。

### 错题 3／13｜が vs は：できる、好き、わかる 前面用が

> **出处：** 三-4
> **危险等级：** 🟡
> **标签：** #助词误用
> **语法点：** [G012] できる

#### 错在哪

三-4　日本語（　）できますか。　~~は~~ → **が**

#### 到底怎么回事

你用は来提示话题，这个思路在很多句子里是对的。但 できる 是特殊的——它的对象固定用が标记。

这不只是できる一个词的事。日语里有一小组动词/形容词，对象都用が：

```
N が できます（会）
N が 好きです（喜欢）
N が わかります（懂）
N が ほしいです（想要）
```

四个词，同一条规则。看到它们，前面就填が。

韩语也有类似现象：「일본어**가** 가능합니까」——가능하다 前面也用 가（主格）而不是 을/를（宾格）。逻辑一样。

#### 练

- [ ] 1. ピアノ（　）できますか。
- [ ] 2. 日本の映画（　）好きです。
- [ ] 3. この漢字（　）わかりますか。

### 错题 4／13｜前に vs までに：动作先后 vs 截止期限

> **出处：** 四-7、七-5
> **危险等级：** 🔴
> **标签：** #句型混淆
> **语法点：** [G009] までに vs まで、[G013] V前に

#### 错在哪

四-7　寝る（　）、歯を磨きます。　~~後で~~ → **前に**
七-5　吃饭之前洗手。　~~ご飯たべまでに手が洗うます~~ → **ご飯を食べる前に、手を洗います。**

#### 到底怎么回事

四-7 是语义理解出了问题：刷牙在睡觉**前面**，不是后面。

七-5 的问题更多。你知道是"之前"，但用了 までに。这两个都能翻成"之前"，区别是：

- **前に**：两个动作有先后。"A前に B"= 先做B再做A。→ 寝る前に、歯を磨く（先刷牙后睡觉）
- **までに**：有一个截止时间点。"X までに Y"= 在X之前完成Y。→ 金曜日までに、レポートを出す（周五是 deadline）

判断方法：前面是**动词**（两个动作的先后）→ 前に。前面是**时间点**（deadline）→ までに。

韩语对照：前に ≈ 기 전에（자기 전에 이를 닦아요），までに ≈ 까지（금요일까지 제출해 주세요）。

#### 对比

| 场景 | 用哪个 | 例句 |
|------|--------|------|
| 睡前刷牙（动作先后） | V原形＋**前に** | 寝る前に歯を磨く |
| 周五前交报告（deadline） | 时间＋**までに** | 金曜日までにレポートを出す |

#### 练

- [ ] 1. 出かける（前に／までに）、窓を閉めてください。
- [ ] 2. 来週の月曜日（前に／までに）、宿題を出してください。
- [ ] 3. 会議が始まる（前に／までに）、資料を読みます。

### 错题 5／13｜を 的两种身份：宾语 vs 经过的场所

> **出处：** 五-3
> **危险等级：** 🟡
> **标签：** #助词误用
> **语法点：** [G002] を离经用法

#### 错在哪

五-3　公園**を**散歩します。→ 问：这里的を是什么用法？　~~宾语~~ → **经过的场所**

#### 到底怎么回事

「公園を散歩する」不是"散步公园（宾语）"，是"在公园里散步（场所）"。

判断标准：看动词。散歩する、歩く、渡る、走る、出る、通る——这些都是**移动类动词**。移动类动词前面的を，标记的是"经过/穿过的地方"，不是宾语。

```
移动动词 ＋ を → 场所：公園を散歩する、橋を渡る、道を歩く
普通动词 ＋ を → 宾语：りんごを食べる、本を読む
```

#### 练

- [ ] 1. 交差点を渡ります — を是宾语还是场所？
- [ ] 2. コーヒーを飲みます — を是宾语还是场所？

### 错题 6／13｜造句时 で/へ/を 全搞混

> **出处：** 七-1
> **危险等级：** 🔴
> **标签：** #助词误用 #中式直译
> **语法点：** 助词 で/へ/を

#### 错在哪

七-1　请不要在这里抽烟。
你写的：~~ここへタバコは吸わないでください~~
正确的：**ここでたばこを吸わないでください。**

两个错：
1. ここ**へ** → ここ**で**。へ是方向（往哪去），で是动作发生的地点（在哪做事）。抽烟这个动作发生在"这里"，用で。
2. タバコ**は** → たばこ**を**。たばこ是"吸"的宾语，用を。

这是中文"在这里"直接对应到へ的典型中式直译。中文的"在"不区分方向和地点，日语区分。

```
在哪做事 → で：ここで吸う、図書館で勉強する
往哪去 → へ：学校へ行く、日本へ来る
动作对象 → を：たばこを吸う、本を読む
```

韩语对照：で ≈ 에서（여기에서 피우다），へ ≈ 에/로（학교에 가다）。跟韩语的 에서/에 区分完全对应。

#### 对比

| 中文 | 错误 | 正确 | 为什么 |
|------|------|------|--------|
| 在这里抽烟 | ここへ | ここ**で** | 做事的地点用で |
| 去学校 | 学校で | 学校**へ** | 方向用へ |
| 抽烟 | タバコは | たばこ**を** | 宾语用を |

#### 练

- [ ] 1. レストラン（　）ご飯を食べます。
- [ ] 2. デパート（　）買い物に行きます。
- [ ] 3. ここ（　）写真（　）撮らないでください。

### 错题 7／13｜レポート 拼写 + 宾语は→を

> **出处：** 七-2
> **危险等级：** 🟢
> **标签：** #助词误用 #词汇混淆
> **语法点：** 外来语拼写、助词を

#### 错在哪

七-2　明天之前必须写完报告。
你写的：~~明日までにレポとは書かなければなりません~~
正确的：**明日までにレポートを書かなければなりません。**

两个小问题：

### 错题 8／13｜"会弹钢琴"：V原形＋ことができます

> **出处：** 七-4
> **危险等级：** 🔴
> **标签：** #句型混淆
> **语法点：** [G012] Vることができる

#### 错在哪

七-4　我会弹钢琴。
你写的：~~私はピアノが弾きあります~~
正确的：**私はピアノを弾くことができます。**

#### 到底怎么回事

「弾きあります」这个结构日语里不存在。你可能把 できる 和 あります 搅到一起了。

"会做某事"在日语里有两种说法：

1. **名词直接接**：N が できます → 日本語ができます（会日语）
2. **动词要加こと**：V原形 ＋ ことができます → ピアノを弾くことができます（会弹钢琴）

动词不能直接接できる。中间必须有「こと」把动词变成名词。

韩语对照：「피아노를 칠 수 있습니다」—— "ㄹ 수 있다" 对应 "ことができる"，结构一模一样。

#### 对比

| 错误 | 正确 | 问题 |
|------|------|------|
| 弾きあります | 弾く**ことができます** | 没有"弾きあります"这种说法 |
| 話しできます | 話す**ことができます** | 动词不能直接接できる |

#### 练

- [ ] 1. 我会开车。→ 車を＿＿＿＿＿＿＿＿。
- [ ] 2. 我会做日本料理。→ 日本料理を＿＿＿＿＿＿＿＿。
- [ ] 3. 他会说英语。→ 彼は英語を＿＿＿＿＿＿＿＿。

### 错题 9／13｜"知道" ≠ "有"：知っています vs あります

> **出处：** 七-8
> **危险等级：** 🔴
> **标签：** #词汇混淆
> **语法点：** 知っています

#### 错在哪

七-8　你知道那家餐厅吗？
你写的：~~あのレストランはありますか~~
正确的：**あのレストランを知っていますか。**

#### 到底怎么回事

你把"知道"翻成了"有/存在"。可能是中文"知不知道"在脑子里变成了"有没有"→ ありますか。

这两个词意思完全不一样：
- **知っています**（知道）：知る的て形＋います，表示"知道"这个状态
- **あります**（有/存在）：东西在某个地方

"你知道那家餐厅吗" = あのレストラン**を知っていますか**
"那附近有餐厅吗" = 近くにレストラン**がありますか**

#### 练

- [ ] 2. 駅の近くにコンビニが（　　　　　）か。（有吗）

### 错题 10／13｜だんだん＋寒くなる：三个地方同时错

> **出处：** 七-9
> **危险等级：** 🔴
> **标签：** #词汇混淆 #变形错误
> **语法点：** [G016] だんだん～なる

#### 错在哪

七-9　渐渐变冷了。
你写的：~~続々寒いなります~~
正确的：**だんだん寒くなりました。**

三个错：
1. ~~続々~~ → **だんだん**。続々（ぞくぞく）是"接二连三、陆续"的意思，比如「人が続々と集まった」（人陆续到了）。"渐渐"是 だんだん。
2. ~~寒い~~ → **寒く**。い形容词接なる，必须把い变成く。寒い → 寒く＋なる。
3. ~~なります~~ → **なりました**。"变冷了"是已经发生的事，要用过去时。

```
だんだん ＋ い形（く形）＋ なる
  だんだん寒くなりました（渐渐变冷了）

だんだん ＋ な形 ＋ になる
  だんだん上手になりました（渐渐变好了）
```

韩语对照：だんだん ≈ 점점，寒くなる ≈ 추워지다。い→く＋なる 跟 韩语 ㅂ불규칙→워지다 的变形逻辑类似。

#### 对比

| 你写的 | 正确的 | 哪里不对 |
|--------|--------|---------|
| 続々 | **だんだん** | 续々是"陆续"不是"渐渐" |
| 寒い**なる** | 寒**く**なる | い→く 再接なる |
| なり**ます** | なり**ました** | 已经变了→过去时 |

#### 练

- [ ] 1. 日本語が（　　　）上手（　）なりました。
- [ ] 2. （　　　）暑（　）なりますね。

### 错题 12／13｜ないでください："请不要"是一整块

> **出处：** 十-2
> **危险等级：** 🟡
> **标签：** #句型混淆
> **语法点：** [G006] ないでください

#### 错在哪

十-2　纠错题：ここで写真を撮ら**なくて**ください
你找到了错误位置。但你改成了「撮ら**なくで**ください」——还是不对。
正确的是：**撮らないでください**。

#### 到底怎么回事

你把「なくて」改成了「なくで」，等于只去掉了一个「て」。但问题不在这。

"请不要做" = **Vない形 ＋ で ＋ ください**

「ない」是一个完整的词。后面直接接「で」。不是「なく＋で」，不是「なく＋て」，就是「ない＋で」。

只有一个正确写法，其他全是错的：
```
✓ ないでください
✗ なくてください
✗ なくでください
✗ ないください
```

#### 练

- [ ] 1. ここで走ら＿＿＿＿ください。
- [ ] 2. 窓を開け＿＿＿＿ ください。
- [ ] 3. 纠错：ここで食べなくてください → ＿＿＿＿＿＿＿＿

### 错题 13／13｜あの人を誰も知りません：知る的宾语用を

> **出处：** 十-5
> **危险等级：** 🟡
> **标签：** #助词误用
> **语法点：** 助词を（宾语）

#### 错在哪

十-5　纠错题：あの人**は**誰も知りません。
你改成了 ~~は→が~~。不对。正确的是 **は→を**。

#### 到底怎么回事

这句话的意思是"谁都不认识那个人"。「あの人」是"知道"的对象——被知道的东西。对象 = 宾语 = を。

你在は不对的时候想到了が，但が是主语标记。这里あの人不是做动作的人，是被动作指向的对象。

知る的宾语用を：あの人**を**知っていますか（认识那个人吗？）

- [ ] 2. 田中さん（　）誰も知りません。

#### 最该先修的三个问题

1. **助词 で/へ/を/が**——造句之前先拆一遍：谁（は）在哪（で）对什么（を/が）做什么。不要从中文"在"直接翻。这次 18 题错了 8 题是助词，占比接近一半。
2. **また/もう**——后面跟数量词就用もう，不跟就用また。就这一条。这次错了 3 题全是同一个坑。
3. **た形特殊变形**——先想て形再换た/だ。行く和来る两个特例死记。这次 3 题变形全错，说明还没形成条件反射。
