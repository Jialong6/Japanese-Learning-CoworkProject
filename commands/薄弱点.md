# 薄弱点 — 查看错题统计和薄弱语法点分析

触发词：`看看错题本`、`我的薄弱点`、`薄弱点`、`哪里不行`

## 上下文

执行前先读取 `contexts/复习模式.md`，切换到复习模式。

## 执行流程

### 第1步：读取数据

1. 读 `错题本.md` → 薄弱语法点汇总 + 所有练习记录
2. 读 `错题精讲.md` → 错误模式分析

### 第2步：多维度统计

**按错误次数排序**（最常错的排在前面）：
- 语法点 | 状态 | 累计错误次数 | 最近是否做对

**按题型统计**（哪类题失分最多）：
- 题型 | 历次平均得分率 | 趋势（↑ 进步 / ↓ 退步 / → 持平）

**按错误类型分类**：
- 助词相关（で/に/を/が/は...）
- 变形相关（て形/た形/ない形...）
- 句型混淆（前に vs までに, また vs もう...）
- 词汇混淆（だんだん vs 続々...）
- 造句综合能力

### 第3步：输出分析

```
## 薄弱点分析（YYYY-MM-DD）

### Top 5 最需要加强的语法点
1. 语法点 — 错X次，状态🔴，核心问题是...
2. ...

### 题型表现
| 题型 | 平均得分率 | 趋势 |
| ... | ... | ... |

### 进步了的地方
- 哪些语法点从🔴变成了🟡或🟢
- 哪些题型得分率提升了

### 针对性建议
- 最需要集中攻克的1-2个问题
- 建议的练习方式
```

## 注意

- 如果只做过一次练习，没有趋势数据，就说"数据还不够，多做几次练习后才能看到趋势"
- 先说进步，再说不足，保持鼓励
