# 出题 — 生成练习卷

触发词：`出题`、`给我出练习`、`练习`

## 上下文

执行前先读取 `contexts/出题模式.md`，切换到出题模式。

## 执行流程

按以下顺序严格执行，不跳步：

### 第1步：读取状态

1. 读 `CLAUDE.md` → 获取当前词汇范围和语法范围
2. 读 `错题本.md` → 获取薄弱语法点列表、状态、下次复习日期
3. 读 `错误模式库.md` → 获取 Jay 的错误思维模式（🔥高频优先）
4. 读 `continuous-learning-config.md` → 获取模式与出题的联动规则
5. 读 `语法知识库.md` → 获取可出题的语法点和例句

### 第2步：确定本次出题范围

1. 检查今天日期，筛选错题本中「下次复习」≤ 今天的语法点 → 这些是**必考点**
2. 统计各语法点的状态：
   - 🔴 未掌握 → 高权重（多出题）
   - 🟡 不稳定 → 中权重
   - 🟢 已掌握 → 低权重（偶尔出现巩固）
3. 如果 Jay 指定了练习范围（如"出16到19课的题"），以指定范围为主，但仍按上述权重分配

### 第3步：组卷

按 CLAUDE.md 中的出题原则组卷：

- 总题量：60-80 小题
- **70% 薄弱点**：优先覆盖所有到期复习的语法点，每个至少出2题
- **20% 巩固**：已掌握内容随机出题
- **10% 拓展**：已学语法的新组合或稍复杂句子
- 从题型菜单中选 5-8 种题型，避免连续两次练习题型完全相同
- 造句/翻译题用生活场景

### 第4步：词汇检查

逐题检查：所有日语词汇是否在 Jay 已掌握范围内（第1課～当前課）。如果某个词超出范围，换成范围内的近义词或重新造题。

### 第5步：生成文件

1. 文件名格式：`语法练习_YYYY-MM-DD_范围.md`
2. 文件开头写明：日期、考察范围、题量、预计用时
3. 每道题留出答题空间（空行或括号）
4. 造句/翻译题写清中文原文
5. 文件末尾写：「做完后告诉我"批改"」

### 第6步：出题后汇报

告诉 Jay：
- 本次练习覆盖了哪些语法点
- 哪些是复习到期的重点
- 预计用时
- 文件路径

### 第7步：模式针对性出题

检查错误模式库中活跃度为 🔥 的模式，确保每个 🔥 模式在本次练习中至少有 2 道题直接测试。使用模式详情中的「针对性出题建议」来设计这些题。

例如：
- P02（宾语用は代替を）🔥 → 出 は/を 对比填空
- P06（中文直译拼接）🔥 → 造句题前加结构分析步骤
- P03（もう/また 混淆）🔥 → 出もう/また 交替场景

## 注意

- 不要在练习卷中暴露答案
- 选择题的选项要有干扰性，不能一眼看出答案
- 纠错题的错误要贴近 Jay 的真实错误模式（参考错误模式库中的「思维路径」）
- 造句题难度递进：简单句 → 复合句
- 🔥 模式的针对性题目要设计得自然，不能让 Jay 感觉在被反复考同一个东西
