# 批改 — 批改练习卷并更新学习记录

触发词：`做完了`、`批改`、`改卷`

## 执行流程

### 第1步：定位练习卷

1. 找到最新的 `语法练习_*.md` 文件
2. 如果有多份未批改的练习卷，问 Jay 批改哪一份
3. 读取练习卷全文，确认 Jay 已经填写了答案

### 第2步：逐题批改

对每一道题：

1. 判定对错，标注 ✅ 或 ❌
2. 错题必须给出：
   - 正确答案
   - 简明解释（1-2句话说清为什么错）
   - 涉及的语法点编号（对应语法知识库的ID）
3. 区分笔误和语法错误：
   - 笔误（打字错误、少一个假名等）→ 标注「笔误」，不计入错题统计
   - 语法错误 → 正常计入
4. 造句/翻译题的评分标准：
   - 语法正确、意思对 → ✅
   - 意思对但有语法错误 → ❌ 但给部分肯定
   - 多个错误的 → 逐个指出

### 第3步：统计得分

```
总分：XX/XX（XX%）

| 题型 | 得分 | 得分率 |
|------|------|--------|
| ... | .../... | ...% |
```

### 第4步：更新错题本

读取 `错题本.md`，按以下规则更新：

1. **新错的语法点**（之前没错过）：
   - 添加到薄弱语法点汇总表
   - 状态设为 🔴 未掌握
   - 错误次数 = 1
   - 下次复习 = 明天

2. **再次做错的语法点**（之前错过）：
   - 错误次数 +1
   - 状态保持 🔴 未掌握
   - 下次复习重置为明天

3. **之前错过但这次做对的**：
   - 状态更新为 🟡 不稳定
   - 按艾宾浩斯间隔推进下次复习日期

4. **连续2次以上做对且间隔>7天的**：
   - 状态更新为 🟢 已掌握

5. 在练习记录区追加本次记录（得分、各题型得分率、错题明细）

### 第5步：更新错题精讲

读取 `错题精讲.md`，追加本次错题的深度讲解：

- 按「事实驱动」风格写（短句、可验证事实、不用模版连接词）
- 每道错题：错在哪 → 规则是什么 → 对比正误 → 变式练习
- 如果有韩语类比能帮助理解的，加上

### 第6步：提取错误模式（continuous-learning）

读取 `错误模式库.md`，对照本次错题，执行模式提取：

1. **匹配已有模式**：这次的错误是否命中了已有的模式（P01-PXX）？
   - 如果命中 → 触发次数 +1，更新「最近触发」日期
   - 如果活跃度变化（比如连续两次都犯）→ 升级为 🔥
   - 如果某个模式这次没犯且距离上次触发 ≥ 14天 → 降级为 💤

2. **发现新模式**：这次的错误中是否有新的思维路径，不属于任何已有模式？
   - 如果有 → 创建新模式条目，包含：
     - 模式名称（简短描述错误思维）
     - 思维路径（Jay 脑子里在想什么）
     - 实际规则
     - 检测信号（什么场景下容易触发）
     - 针对性出题建议
   - 新模式编号递增：P09, P10...

3. **更新总览表和更新日志**

### 第7步：输出批改反馈

按 CLAUDE.md 中的批改反馈格式输出：

1. 得分总结
2. 错题详解（逐题）
3. 薄弱点总结（这次暴露的共性问题）
4. **模式库更新摘要**（这次命中了哪些模式、是否发现新模式）
5. 下次练习建议（时间 + 重点内容）

### 第8步：提醒

告诉 Jay 下次复习时间和重点。语气鼓励。
