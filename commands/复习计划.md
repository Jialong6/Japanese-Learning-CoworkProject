# 复习计划 — 查看艾宾浩斯复习安排

触发词：`复习计划`、`该复习什么了`、`今天练什么`

## 上下文

执行前先读取 `contexts/复习模式.md`，切换到复习模式。

## 执行流程

### 第1步：读取数据

1. 读 `错题本.md` → 获取所有薄弱语法点及其复习日期
2. 获取今天日期

### 第2步：分类整理

把所有语法点按状态和时间分为四组：

**A. 今天到期** — 下次复习日期 ≤ 今天
**B. 即将到期** — 下次复习日期在未来3天内
**C. 远期计划** — 下次复习日期在3天以后
**D. 已掌握** — 🟢 状态，暂不需要复习

### 第3步：输出报告

```
## 复习计划（YYYY-MM-DD）

### 今天必须复习（X个语法点）
- 语法点名称 | 状态 | 已错X次 | 上次复习距今X天
- ...

### 3天内到期（X个语法点）
- 语法点名称 | 状态 | 预计复习日期
- ...

### 远期计划
- 语法点名称 | 预计复习日期
- ...

### 已掌握（X个）
- 简要列出

### 建议
- 今天是否需要练习
- 如果需要，建议练多少题、侧重什么
- 如果不需要，下次练习时间是什么时候
```

## 注意

- 如果今天没有到期的复习，也要告诉 Jay "今天没有到期复习，最近一次是XX日期"
- 如果积压了很多到期未复习的，提醒 Jay 尽快安排，但不要有压力感
