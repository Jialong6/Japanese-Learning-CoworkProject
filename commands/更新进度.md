# 更新进度 — 更新学习者词汇量和语法范围

触发词：`更新词汇量到第X课`、`我学到第X课了`、`更新进度`

## 执行流程

### 第1步：确认范围

1. Jay 说了具体课数 → 直接使用
2. Jay 没说具体课数 → 问清楚：词汇学到第几课？语法学到第几课？（可以不一样）

### 第2步：更新 CLAUDE.md

修改学习者档案中的两个字段：

- `当前词汇量` → 更新到新的课数和估算词汇量
- `已学语法范围` → 更新到新的课数

词汇量估算规则（大家的日本语 初级Ⅰ 每课约40-50词）：
- 第1-25课 ≈ 1100词
- 第26-50课 ≈ 另外1100词
- 粗略估算：课数 × 44

### 第3步：检查新增语法

如果语法范围扩大了：

1. 检查 `语法知识库.md` 是否已包含新课的语法点
2. 如果没有 → 提醒 Jay 提供新课的语法笔记，或由 Claude 根据教材补充
3. 如果有 → 告诉 Jay 新增了哪些语法点，下次出题会覆盖

### 第4步：确认输出

```
已更新：
- 词汇范围：第1課～第XX課（约XXXX词）
- 语法范围：第1課～第XX課
- 新增语法点：列出新课的语法点（如有）

下次出题将包含新范围的内容。
```
