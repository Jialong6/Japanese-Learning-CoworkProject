## G100：ある vs いる（存在表达）

> **来源课：** 第10課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #存在 #ある #いる #第10課

**核心结构：**

```
ある ＝ 表示事物、植物等不会自己移动的主体の存在
いる ＝ 表示人、动物等会自己移动的主体の存在

场所 に 主体 が あります/います
```

**中文含义：** "有"或"在"。关键区分は主体能否自己移动。

**使用场景：**
1. 事物存在：机の上に本があります。（桌上有书。）
2. 人/动物存在：私のうちに犬がいます。（我家有狗。）

**例句库：**
- 私のうちに犬がいます。（我家有狗。）〔笔记〕
- 机の上に本があります。（桌上有书。）〔自造〕
- 公園に子どもがいます。（公园里有孩子。）〔自造〕

**易错点：**
- ⚠️ 植物用「ある」（不会自己移动）：庭に木があります ✓
- ⚠️ 「人がいる」不是「人がある」

**关联对比：**
- [G101] に vs で（存在 vs 动作の场所）

## G101：に vs で（存在场所 vs 动作场所）

> **来源课：** 第10課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #助词 #に #で #场所 #第10課

**核心结构：**

```
に ＝ 存在の场所（ある/いる 搭配）
で ＝ 动作发生の场所（一般动词搭配）
```

**中文含义：** 都可以翻译成"在"，但功能不同。「に」标记"在哪里存在"，「で」标记"在哪里做事"。

**使用场景：**
1. 存在：私のうち**に**犬がいます。（我家有狗。）→ 狗存在于我家
2. 动作：私のうち**で**犬が走ります。（狗在我家跑。）→ 跑这个动作发生在我家

**例句库：**
- 私のうちに犬がいます。（我家有狗。）〔笔记〕
- 私のうちで犬が走ります。（狗在我家跑。）〔笔记〕

**易错点：**
- ⚠️ 同一个场所，搭配 ある/いる 时用「に」，搭配动作动词时用「で」
- ⚠️ 这是 P01 错误模式（中文"在"直译）の根源之一

**关联对比：**
- [G002] を的离经用法 — 移动动词の场所用「を」
- に还有其他用法：时间/对象/目的地（见 G105）

## G102：～や～など（举例列举）

> **来源课：** 第10課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #助词 #列举 #第10課

**核心结构：**

```
AやBなど ＝ A和B之类的（举例，不完全列举）

三种写法意思相同：
1. 机や椅子などがあります。
2. 椅子などがあります。
3. 机や椅子があります。
```

**中文含义：** "…和…之类的"。「や」连接举例项目，「など」表示"等等"。可以省略其中一个。

**例句库：**
- 机や椅子などがあります。（有桌子和椅子之类的。）〔笔记〕
- りんごやみかんを買いました。（买了苹果和橘子之类的。）〔自造〕

**易错点：**
- ⚠️ 「と」是全部列举（A和B），「や」是部分列举（A和B等）
- ⚠️ 「など」可以只跟在最后一个名词后面

**关联对比：**
- と（完全列举：AとB＝A和B，就这两个）
- や…など（部分列举：AやBなど＝A和B之类的，可能还有别的）

## G104：に的多种用法

> **来源课：** 第11課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #助词 #に #多义 #第11課

**核心结构：**

```
1. 时间：  月曜日に会います。（星期一见面。）
2. 对象：  友達に電話をかけます。（给朋友打电话。）
3. 存在场所：東京に住んでいます。（住在东京。）
4. 频率：  1日に3回飲みます。（一天吃三次。）
```

**中文含义：** 「に」是用法最多の助词之一。时间点、对象、存在场所、频率都用「に」。

**使用场景：**
1. 频率表达：この薬は1日に3回飲みます。（这药一天吃三次。）

**例句库：**
- この薬は1日に3回飲みます。（这药一天吃三次。）〔笔记〕
- 日曜日に友達に会います。（星期天和朋友见面。）〔自造〕

**易错点：**
- ⚠️ 频率の「に」容易忘写：1日**に**3回 ✓ / 1日3回 也可（に可省略）
- ⚠️ 注意同一句中可能出现多个「に」，各有不同含义

**关联对比：**
- [G101] に vs で — 场所用法の区分
- 第15课会进一步补充 に 的"附着点"用法

## G106：ごろ vs ぐらい（大约）

> **来源课：** 第11課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #时间 #近义 #第11課

**核心结构：**

```
ごろ  ＋ 时间点 ＝ ～点左右    例：七時半ごろです。（七点半左右。）
ぐらい ＋ 时间段 ＝ ～长左右   例：一時間半ぐらいです。（一个半小时左右。）
```

**中文含义：** 都表示"大约"，但「ごろ」接时间点，「ぐらい」接时间段/数量。

**易错点：**
- ⚠️ 时间**点**用「ごろ」：7時**ごろ** ✓ / 7時**ぐらい** ✗
- ⚠️ 时间**段**用「ぐらい」：2時間**ぐらい** ✓ / 2時間**ごろ** ✗

**例句库：**
- 七時半ごろです。（七点半左右。）〔笔记〕
- 一時間半ぐらいです。（一个半小时左右。）〔笔记〕

## G107：时态体系まとめ（動詞/名詞・二類形/一類形）

> **来源课：** 第12課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #时态 #変形 #まとめ #第12課

**核心结构：**

```
【動詞谓语句】
  非过去 肯定：Vます         非过去 否定：Vません
  过去 肯定：Vました         过去 否定：Vませんでした

【名詞・二類形容词 谓语句】
  非过去 肯定：です          非过去 否定：ではありません
  过去 肯定：でした          过去 否定：ではありませんでした

【一類形容词（い形）谓语句】
  非过去 肯定：～いです       非过去 否定：～くないです
  过去 肯定：～かったです     过去 否定：～くなかったです
```

**中文含义：** 三套时态变化体系。動詞用ます系统，名詞/な形用です系统，い形用自己的变化。

**易错点：**
- ⚠️ い形容词の过去肯定是「かったです」不是「いでした」：寒**かったです** ✓ / 寒い**でした** ✗
- ⚠️ い形容词の否定是「くないです」不是「くありません」（后者也可，但初学用前者）
- ⚠️ 「いい」の变形特殊：よかったです / よくないです / よくなかったです

**关联对比：**
- [G016] だんだん～なる — 形容词变形和「なる」前の接续相关

## G108：～より（比较）＋ ～のほうが ＋ 一番

> **来源课：** 第12課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #比较 #句型 #第12課

**核心结构：**

```
A は B より ～です      ＝ A比B更～
  例：バナナはりんごより甘いです。（香蕉比苹果甜。）

AとBと どちらが ～ですか  ＝ A和B哪个更～？（二选一）
  答：Aのほうが ～です      ＝ A比较～

（三个以上）～の中で 何/どれ/どこ が一番 ～ですか  ＝ ～中哪个最～？
```

**中文含义：** 日语比较句の三种基本形式：比较、二选一、最高级。

**使用场景：**
1. 比较：東京は大阪より大きいです。（东京比大阪大。）
2. 二选一：お茶とコーヒーとどちらが好きですか。——お茶のほうが好きです。
3. 最高级：果物の中でどれが一番好きですか。——りんごが一番好きです。

**易错点：**
- ⚠️ 二选一用「どちら」，三个以上用「どれ/何/どこ」
- ⚠️ 回答二选一固定用「～のほうが」

**关联对比：**
- [G103] 疑问词まとめ — どちら/どれ の使い分け

**踩坑记录：**
（尚未练习）

## G109：で/との用法拡張

> **来源课：** 第12課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #助词 #で #と #第12課

**核心结构：**

```
で の三种用法：
  1. 方式・手段 "用"：箸で食べます。（用筷子吃。）
  2. 动作场所 "在"：図書館で勉強します。（在图书馆学习。）
  3. 范围 "在～中"：日本で一番高い山は富士山です。（在日本最高的山是富士山。）

と の两种用法：
  1. 和：友達と行きます。（和朋友去。）
  2. 比较句の列举：AとBとどちらが～（A和B哪个更～）
```

**易错点：**
- ⚠️ 「で」的范围用法容易和场所用法混淆——范围「で」常出现在比较/最高级句型中

**关联对比：**
- [G101] に vs で（存在 vs 动作场所）
- [G102] や…など（部分列举）vs と（完全列举/比较列举）

## G110：欲しい / Vたい（想要/想做）

> **来源课：** 第13課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #欲望 #たい #い形容词 #第13課

**核心结构：**

```
N が 欲しい     ＝ 想要N     例：車が欲しいです。（想要车。）
V ます形 ＋ たい ＝ 想做～    例：食べたいです。（想吃。）

注意：「たい」作为い形容词活用
  否定：食べたくないです
  过去：食べたかったです
```

**中文含义：** 表达欲望。「欲しい」用于名词（想要东西），「Vたい」用于动词（想做事）。

**使用场景：**
1. 想要东西：新しいパソコンが欲しいです。（想要新电脑。）
2. 想做事：日本へ行きたいです。（想去日本。）

**易错点：**
- ⚠️ 「欲しい」前面の助词是「**が**」不是「を」：車**が**欲しい ✓
- ⚠️ 「たい」は い形容词，活用方式和い形容词一样
- ⚠️ 「たい」通常只用于第一人称（自己），问别人想不想用「～たいですか」

**关联对比：**
- [G012] できる — 能力（能做）vs たい — 愿望（想做）

**🇰🇷 韩日类比：**
- 「たい」≈ 韩语的 「-고 싶다」，活用方式也类似形容词

## G111：疑问词＋か（不确定）

> **来源课：** 第13課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #疑问词 #不确定 #第13課

**核心结构：**

```
疑问词 ＋ か ＝ 某～（不确定）

どこか    某个地方     ≈ 어딘가
いつか    某个时候     ≈ 언젠가
誰か      某个人      ≈ 누군가
何か      某个东西
```

**中文含义：** 疑问词后面加「か」，从"问题"变成"不确定的某一个"。

**使用场景：**
1. 疑问：誰がいますか。（**谁**在？）→ 想知道是谁
2. 不确定：誰かいますか。（**有人**在吗？）→ 不关心是谁，只问有没有人

**例句库：**
- 誰がいますか。→ 谁在？（누구가 있습니까?）〔笔记〕
- 誰かいますか。→ 有人在吗？（누군가 있습니까?）〔笔记〕
- どこかへ行きたいです。（想去某个地方。）〔自造〕

**🇰🇷 韩日类比：**
- 日语の疑问词＋か 和韩语の疑问词＋ㄴ가 结构完全对应

## G112：が的用法拡張

2. 好恶/欲望の对象：日本語が好きです。/ 車が欲しいです。

## G113：V ます形（去ます）＋に 行く/来る/帰る（目的）

> **来源课：** 第13課 ｜ **添加：** 2026-04-02 ｜ **最后复习：** — ｜ **状态：** 🟡 练习中

**标签：** #句型 #目的 #移动 #第13課

**核心结构：**

```
V ます形（去掉ます）＋ に ＋ 行きます/来ます/帰ります
N ＋ に ＋ 行きます/来ます/帰ります

＝ 去/来/回去 做某事
```

**中文含义：** 表示移动の目的。"去做某事""来做某事"。

**使用场景：**
1. スーパーへ果物を買いに行きます。（去超市买水果。）
2. 図書館へ本を読みに行きます。（去图书馆看书。）

**例句库：**
- スーパーへ果物を買いに行きます。（去超市买水果。）〔笔记〕
- 日本へ日本語を勉強しに来ました。（来日本学日语。）〔自造〕

**易错点：**
- ⚠️ 是ます形**去掉ます**后＋に，不是原形＋に：買い**に**行く ✓ / 買う**に**行く ✗
- ⚠️ 名词也可以直接接「に」：買い物**に**行く（去购物）

**关联对比：**
- [G013] V原形＋前に — 都用「に」但接续不同（前に 接原形，目的の に 接ます形去ます）

**🇰🇷 韩日类比：**

## G116：て形句型拡張

```
1. Vてください         ＝ 请做～（同第14课）

4. Vてはいけません      ＝ 不可以做～（禁止）
```

**中文含义：**
- 「てもいいですか」：请求许可（"可以～吗？"）

## G001：形容词・名词的て形（并列）

> **来源课：** 第16課 ｜ **添加：** 2026-04-01 ｜ **最后复习：** 2026-04-01 ｜ **状态：** 🟢 已掌握

**标签：** #变形 #形容词 #名词 #并列 #第16課

**核心结构：**

```
一类形容词（い形）：い → くて       例：高い → 高くて
二类形容词（な形）：词干 ＋ で      例：静か → 静かで
名词：             N ＋ で        例：学生 → 学生で
```

**中文含义：** 表示并列（"又…又…"），连接两个以上的形容词或名词描述。与动词て形表顺序不同，形容词/名词的て形表并列。

**使用场景：**
1. 描述一个事物的多个特征：この部屋は広くて、明るいです。（这个房间又大又亮。）
2. 描述一个人的多个特点：ミラーさんは元気で、親切です。（米勒先生精力充沛又亲切。）

**例句库：**
- 大阪は賑やかで、食べ物が美味しいです。（大阪很热闹，食物很好吃。）〔教材〕
- あの先生は若くて、親切です。（那位老师年轻又亲切。）〔自造〕
- 彼女は学生で、20歳です。（她是学生，20岁。）〔自造〕

**易错点：**
- ⚠️ 二类形容词和名词都用「で」，不是「くて」。**静かくて ✗ → 静かで ✓**
- ⚠️ 「いい」的て形是「よくて」，不是「いくて」

**关联对比：**
- [G002] 动词て形 — 动词的て形表顺序，不表并列
- い形容词否定て形：高くない → 高くなくて

**踩坑记录：**
- 练习中第十题-1纠错：正确识别了「静かくて→静かで」的错误 ✅（2026-04-01）

## G002：を的离经用法（离开・经过的场所）

> **来源课：** 第16課 ｜ **添加：** 2026-04-01 ｜ **最后复习：** 2026-04-01 ｜ **状态：** 🟡 练习中

a. 离开：電車を降ります（下电车）、大学を出ます（大学毕业/离开大学）

## G003：どれ vs どの＋名词

> **来源课：** 第16課 ｜ **添加：** 2026-04-01 ｜ **最后复习：** 2026-04-01 ｜ **状态：** 🟢 已掌握

**核心结构：**

```
どれ    ＝ 哪个（独立使用，三个及以上选择，不能指人）
どの＋N ＝ 哪个～（修饰名词，三个及以上选择）
```

- これ／それ／あれ → どれ（独立指物）

## G004：もう・まだ・また・も

> **来源课：** 第16課 ｜ **添加：** 2026-04-01 ｜ **最后复习：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #副词 #易混淆 #高频 #第16課

**核心结构：**

```
もう ＋ V过去式     ＝ 已经～了     例：もう帰りました（已经回去了）
もう ＋ 数量词      ＝ 再～一个     例：もう一杯（再来一杯）

まだ               ＝ 还没、尚未    例：まだです（还没）
まだ ＋ V-ています  ＝ 还在～       例：まだ食べています（还在吃）

また               ＝ 再次、又      例：また来てください（请再来）

も（名词后）        ＝ 也           例：私も行きます（我也去）
も（疑问词后＋否定） ＝ 都不/没      例：誰もいません（谁都不在）
```

**中文含义：** 四个词都跟"再/还/也"有关但用法完全不同。最容易混的是「もう＋数量词（再来一个）」和「また（再次）」。

**使用场景：**
1. 已经：もう6時ですね。（已经6点了。）
2. 再来一个：お茶をもう一杯いかがですか。（再来一杯茶怎么样？）
3. 下次再来：また来てください。（请再来。）
4. 还没：A：宿題はできましたか。B：いいえ、まだです。（还没。）

**例句库：**
- もう帰りましたか。——いいえ、まだです。（已经回去了吗？——不，还没。）〔教材〕
- 楽しかったです。また行きたいです。（很开心。还想再去。）〔自造〕
- 日本語も英語もできます。（日语和英语都会。）〔练习〕

**易错点：**
- ⚠️ **もう一杯 vs また来る**：「もう」＋数量词＝再来一个；想表达"下次再～"用「また」
- ⚠️ 「また」不能接数量词。「また一杯」✗ →「もう一杯」✓
- ⚠️ 「も」在疑问词后面表示"都"（全部否定/全部肯定），不是"也"

**关联对比：**
- もう＋数量词（再） vs また（再次）— 这是最高频错题点
- まだ（还没） vs もう（已经）— 常成对出现

**踩坑记录：**
- ❌ 把「もう一杯」错选为「また」（2026-04-01）
- ❌ 把「また来てください」错选为「も」（2026-04-01）
- ❌ 把「また行きたい」错选为「もう」（2026-04-01）

**易错点：**
- ⚠️ 变形基础は ない形，去掉「ない」再接「なければなりません」

**关联对比：**
- [G008] なくてもいいです（不…也行）— 意思相反
- [G006] ないでください（请不要做）— 同为ない形句型

## G009：までに vs まで

> **来源课：** 第17課 ｜ **添加：** 2026-04-01 ｜ **最后复習：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #助词 #时间 #易混淆 #第17課

**核心结构：**

```
までに ＋ 瞬间动词 ＝ 在…之前（截止期限）
まで  ＋ 持续动词 ＝ 一直到…为止（持续终点）
```

**中文含义：** 「までに」强调の是"截止日期"，动作在那之前の某一刻完成；「まで」强调の是"一直持续到那个时间点"。

**使用场景：**
1. までに：土曜日**までに**レポートを**出します**。（星期六之前交报告。）→ "交"は瞬间动作
2. まで：土曜日**まで**レポートを**書きます**。（写报告一直写到星期六。）→ "写"は持续动作

**例句库：**
- 日曜日までに本を返してください。（请在星期天之前还书。）〔練習〕
- 5時まで働きます。（工作到5点。）〔教材〕

**易错点：**
- ⚠️ 关键看后面の动词：瞬间动作（出す、返す、来る）→ までに；持续动作（書く、働く、待つ）→ まで
- ⚠️ 不要和「V原形＋前に」搞混——前に表示"在做某事之前"，までに表示"在某个时间点之前"

**关联对比：**
- [G012] V原形＋前に — 「前に」前面接动词，「までに」前面接时间名词
- から～まで：从…到…

**踩坑记录：**
- ❌ 造句题中把"吃饭之前洗手"用了「までに」，应该用「V原形＋前に」（2026-04-01）

敬体肯定  →  简体肯定（原形）

> **来源课：** 第18課 ｜ **添加：** 2026-04-01 ｜ **最后復習：** 2026-04-01 ｜ **状态：** 🟢 已掌握

## G012：できる（能/会/可以）

> **来源课：** 第18課 ｜ **添加：** 2026-04-01 ｜ **最后復習：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #句型 #能力 #高频 #第18課

**核心结构：**

```
N        ＋ が ＋ できる   例：日本語ができる（会日语）
V原形こと ＋ が ＋ できる   例：ピアノを弾くことができる（会弹钢琴）
```

**中文含义：** 表示有能力做某事、某事是可能の。

**使用场景：**
1. 语言能力：日本語ができます。（会日语。）
2. 技能：ピアノを弾くことができます。（会弹钢琴。）

**例句库：**
- 日本語ができますか。（你会日语吗？）〔教材〕
- 日本語の新聞を読むことができません。（看不了日语报纸。）〔練習〕

**易错点：**
- ⚠️ できる前面の助词は「**が**」，不是「は」也不是「を」
- ⚠️ 动词用法必须は「V原形＋**こと**＋が＋できる」，不能省略「こと」
- ⚠️ 不是「弾きあります」也不是「弾きできます」→ 正确：弾く**ことができます**

**特例/固定搭配：**
- 名词直接接：日本語**が**できる（不需要「こと」）
- 运转/驾驶：車の運転**が**できます

**关联对比：**
- 知っています（知道）vs できます（会/能）— 不要混淆

**踩坑记录：**
- ❌ 「我会弹钢琴」写成了「ピアノが弾きあります」，应为「ピアノを弾くことができます」（2026-04-01）
- ❌ 填助词「日本語（）できますか」填了「は」，应为「が」（2026-04-01）

## G013：V原形＋前に（在做…之前）

> **来源课：** 第18課 ｜ **添加：** 2026-04-01 ｜ **最后復習：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #句型 #时间 #易混淆 #第18課

**核心结构：**

```
V原形         ＋ 前に ＝ 在做…之前
N の（数量词） ＋ 前に ＝ 在…之前

例：寝る前に、歯を磨きます。（睡觉之前刷牙。）
例：食事の前に、手を洗います。（吃饭之前洗手。）
```

**中文含义：** 表示在某个動作發生之前。注意「前に」前面の動詞必須用**原形**，不管整個句子は過去式還是現在式。

**使用場景：**
1. 日常習慣：寝る前に、本を読みます。（睡前看书。）
2. 具體事件：日本へ来る前に、日本語を勉強しました。（来日本之前学了日语。）

**例句库：**
- ご飯を食べる前に、手を洗います。（吃饭前洗手。）〔自造〕
- 寝る前に、歯を磨きます。（睡前刷牙。）〔教材〕

**易错点：**
- ⚠️ 「前に」前面用**V原形**，不是ます形：寝る前に ✓ ／ 寝ます前に ✗
- ⚠️ 不要和「までに」搞混！「前に」前接**动词**，「までに」前接**时间名词**
  - 食べる**前に**（吃之前）vs 金曜日**までに**（星期五之前）

**关联对比：**
- [G009] までに vs まで — 都含有"之前"の意思，但接续不同
- ～てから（做完…之后）— 顺序相反

**踩坑记录：**
- ❌ 选择题把「寝る前に歯を磨きます」选成了「後で」（2026-04-01）
- ❌ 造句题把"吃饭之前洗手"写成「ご飯たべまでに…」，应為「ご飯を食べる前に…」（2026-04-01）

## G015：た形の変形规则

> **来源课：** 第19課 ｜ **添加：** 2026-04-01 ｜ **最后復習：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #变形 #た形 #动词 #高频 #第19課

**核心结构：**

```
规则：把て形の「て」→「た」、「で」→「だ」

て形          →  た形
書いて        →  書いた
飲んで        →  飲んだ
食べて        →  食べた
行って        →  行った     ⚠️ 行く は特殊促音便
来て（きて）   →  来た（きた） ⚠️ V3特殊
して          →  した       ⚠️ V3特殊
遊んで        →  遊んだ     ⚠️ ぶ结尾＝拨音便
話して        →  話した
```

**中文含义：** 動詞の過去簡体形式，表示"做了某事"。變形方法最簡單の記法就は"て→た、で→だ"。

**易错点：**
- ⚠️ **行く→行った**（不は いいた）：行く の て形 は 行って（促音便），所以 た形 は 行った
- ⚠️ **来る→来た（きた）**（不は くった）：三类动词要单独记
- ⚠️ **遊ぶ→遊んだ**（不は 遊ぶんだ）：ぶ结尾は拨音便，て形は遊んで→た形は遊んだ
- ⚠️ 记忆窍门：**先想て形，然后把て换成た就行了**

**关联对比：**
- て形 — た形就は て形 の 「て→た、で→だ」版本
- [G005] ない形 — 另一种重要变形

**踩坑记录：**
- ❌ 行きます→いいた，正确は行った（2026-04-01）
- ❌ 来ます→くった，正确は来た/きた（2026-04-01）
- ❌ 遊びます→遊ぶんだ，正确は遊んだ（2026-04-01）

## G016：だんだん～なる（渐渐变得…）

> **来源课：** 第19課 ｜ **添加：** 2026-04-01 ｜ **最后復習：** 2026-04-01 ｜ **状态：** 🔴 易混淆

**标签：** #句型 #变化 #なる #第19課

**核心结构：**

```
い形容词：い → く ＋ なる     例：寒い → 寒くなる（变冷）
な形容词：词干 ＋ に ＋ なる   例：元気 → 元気になる（变得精神）
名词：    N ＋ に ＋ なる     例：医者 → 医者になる（成为医生）

搭配副词：
だんだん ＝ 渐渐    例：だんだん寒くなりました。（渐渐变冷了。）
```

**中文含义：** 表示状態の変化——"变得…"或"成为…"。

**易错点：**
- ⚠️ い形容词要去掉「い」加「く」再接「なる」：寒**く**なる ✓ ／ 寒**い**なる ✗
- ⚠️ 「渐渐」は「**だんだん**」，不是「续々（ぞくぞく）」。続々＝接连不断（指具体事物一个接一个来）
- ⚠️ 已经发生の変化用过去式「なりました」

**例句库：**
- だんだん寒くなりました。（渐渐变冷了。）〔自造〕
- 日本語が上手になりました。（日语变好了。）〔教材〕

**关联对比：**
- する（主动改变）vs なる（自然变化）：部屋を綺麗にする（打扫房间）vs 部屋が綺麗になる（房间变干净了）

**踩坑记录：**
- ❌ "渐渐变冷了"写成「续々寒いなります」，三处错误：续々→だんだん、寒い→寒く、なります→なりました（2026-04-01）

# 补充：造句常见助词用法速查

> 这不是单独语法点，而是从你の造句错题中归纳の助词速查表。

```
场所助词：
  で ＝ 动作发生の地点      ここで食べます（在这里吃）
  に ＝ 存在/目的地/时间点   東京に住んでいます（住在东京）
  へ ＝ 方向（可和にの互换）   学校へ行きます（去学校）

对象助词：
  を ＝ 宾语（动作对象）     本を読みます（读书）
  を ＝ 离开/经过の场所      電車を降ります（下电车）
  に ＝ 给予/接触の对象      友達に電話をかけます（给朋友打电话）

主题/主语：
  は ＝ 大主题              私は学生です（我是学生）
  が ＝ 小主语/能力对象      髪が長い（头发长）、日本語ができる（会日语）
```

**踩坑记录：**
- ❌ 「ここで」写成「ここへ」（2026-04-01）
- ❌ 「たばこを」写成「タバコは」（2026-04-01）
- ❌ 「手を洗う」写成「手が洗う」（2026-04-01）
- ❌ 「日本語が」写成「日本語は」（2026-04-01）
