# 交互约定（Commands 系统）

每个命令对应 `commands/` 目录下的一个 `.md` 文件，包含详细的执行流程。收到触发词时，先读取对应 command 文件，再按流程执行。

| 触发词 | Command 文件 | 上下文 | 说明 |
|--------|-------------|--------|------|
| `出题` / `给我出练习` | `commands/出题.md` | `contexts/出题模式.md` | 根据错题本和复习计划生成新练习卷 |
| `做完了` / `批改` | `commands/批改.md` | `contexts/批改模式.md` | 逐题批改、更新错题本和错题精讲 |
| `复习计划` / `今天练什么` | `commands/复习计划.md` | `contexts/复习模式.md` | 按艾宾浩斯间隔给出复习安排 |
| `看看错题本` / `我的薄弱点` | `commands/薄弱点.md` | `contexts/复习模式.md` | 汇报错题统计和薄弱点分析 |
| `更新词汇量到第X课` / `我学到第X课了` | `commands/更新进度.md` | — | 更新词汇量和语法范围 |
| `记住这个发现` / `记录一下` | `commands/学习记录.md` | `contexts/讲解模式.md` | 即时记录学习发现 |
| 提供新的语法笔记文件 | — | — | 读取并纳入出题范围 |

> **执行原则：**
> 1. 收到触发词后，先读取对应的 command 文件获取完整流程
> 2. 如果 command 文件指定了上下文文件，也要读取对应的 `contexts/` 文件
> 3. 再严格按步骤执行。不凭记忆执行，每次都读文件。
