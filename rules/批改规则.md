# 批改规则

每次 Jay 做完练习并告知后：

1. **逐题批改** — 对每题标注 ✅ 或 ❌，错题给出正确答案和简明解释
2. **打分** — 统计总分和各题型得分率
3. **错题归类** — 按语法点分类，记录到 `错题本.md`
4. **薄弱点分析** — 总结这次暴露出的主要问题，给出针对性建议
5. **模式提取** — 从错题中提取/更新错误模式（参考 `continuous-learning-config.md`）
6. **更新复习计划** — 在错题本中更新下次复习时间和重点

## 批改反馈格式

```
## 批改结果：语法练习_XXXX

### 得分
总分：XX/XX（XX%）
各题型：...

### 错题详解
（逐题讲解）

### 薄弱点总结
（归纳共性问题）

### 模式库动态
（命中了哪些模式、新发现了什么、活跃度变化）

### 下次练习建议
（重点复习什么、建议什么时候做下一次练习）
```

## 判定标准

- 语法正确 + 意思对 → ✅
- 意思对但语法有误 → ❌（给部分肯定）
- 笔误（打字错误、少一个假名等）→ 标注「笔误」，不计入统计
- 造句题多个错误 → 逐个指出
- 未作答 → 标注「未作答」，不计分
