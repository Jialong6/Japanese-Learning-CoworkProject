# 艾宾浩斯复习计划规则

## 复习间隔

新学/新错的语法点，按以下间隔安排复习：

| 轮次 | 间隔 | 说明 |
|------|------|------|
| 第1次 | 当天 | 出错后立即在反馈中讲解 |
| 第2次 | 1天后 | 下一次练习重点考察 |
| 第3次 | 3天后 | 再次出现在练习中 |
| 第4次 | 7天后 | 一周后复习 |
| 第5次 | 14天后 | 两周后复习 |
| 第6次 | 30天后 | 一个月后复习，通过则标记为"已掌握" |

## 状态定义

- **🔴 未掌握**：错误2次及以上，或最近一次练习仍错
- **🟡 不稳定**：错过但最近一次做对了
- **🟢 已掌握**：连续2次以上做对，且间隔超过7天

## 状态转换规则

| 原状态 | 条件 | 新状态 |
|--------|------|--------|
| 无 → | 首次做错 | 🔴 |
| 🔴 → | 做对一次 | 🟡 |
| 🔴 → | 再次做错 | 🔴（复习日期重置为明天）|
| 🟡 → | 再次做错 | 🔴 |
| 🟡 → | 做对且间隔>7天 | 🟢 |
| 🟢 → | 做错 | 🟡（重新进入复习循环）|
