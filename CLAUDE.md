# 日语学习和练习 项目指令

## 项目定位

这是 Jay 的日语语法练习项目。核心功能：根据艾宾浩斯记忆曲线，有计划地出题、批改、记录错题、追踪薄弱语法点，并动态调整复习计划。

---

## 学习者档案

- **姓名：** Jay
- **母语背景：** 朝鲜族，韩语为母语之一
- **当前词汇量：** 大家的日本语 初级Ⅰ 第1課～第25課（约1100词）
- **已学语法范围：** 第1課～第19課
- **语法知识库：** `语法知识库.md`（按课整理，含结构、例句、易错点、踩坑记录）
- **词汇来源文件：** `大家的日本语初级_全单词表.md`（全50课，出题时只使用已掌握词汇范围内的词）

> **重要：** 出题时，只使用 Jay 已掌握词汇范围内的单词。当 Jay 说"我学到第X课了"或"更新词汇量"时，更新上面的词汇量和语法范围记录。

## 项目结构

### 核心数据文件

| 文件 | 用途 |
|------|------|
| `大家的日本语初级_全单词表.md` | 全单词表（按课、词性结构化），出题素材库 |
| `语法知识库.md` | 语法点知识体系（按课整理，含结构、例句、易错点、踩坑记录） |
| `错题本.md` | 错题记录、薄弱点追踪、复习计划 |
| `错题精讲.md` | 每次练习后的错题深度讲解＋变式强化练习 |
| `错误模式库.md` | Jay 的错误思维模式库（自动从每次批改中提取） |
| `语法练习_*.md` | 各次练习卷 |

### 规则（`rules/` 目录）

详细规则已拆分为独立文件，按需加载：

| 规则文件 | 内容 |
|---------|------|
| `rules/出题规则.md` | 题型菜单、出题原则、题量分配 |
| `rules/批改规则.md` | 批改流程、反馈格式、判定标准 |
| `rules/艾宾浩斯规则.md` | 复习间隔、状态定义、状态转换 |
| `rules/交互约定.md` | Commands 系统、触发词映射、上下文切换 |
| `rules/git工作流.md` | 自动 commit/push 规则 |
| `rules/注意事项.md` | 语言语气、笔误处理、韩日类比 |

### 命令（`commands/` 目录）

| 触发词 | 文件 | 说明 |
|--------|------|------|
| `出题` / `给我出练习` | `commands/出题.md` | 生成新练习卷 |
| `做完了` / `批改` | `commands/批改.md` | 批改 + 更新所有记录 |
| `复习计划` / `今天练什么` | `commands/复习计划.md` | 艾宾浩斯复习安排 |
| `看看错题本` / `我的薄弱点` | `commands/薄弱点.md` | 错题统计和薄弱点分析 |
| `更新词汇量到第X课` | `commands/更新进度.md` | 更新学习范围 |
| `记住这个发现` / `记录一下` | `commands/学习记录.md` | 即时记录学习发现 |

### 上下文模式（`contexts/` 目录）

不同命令激活不同的工作模式，定义行为优先级和关注点：

| 模式 | 文件 | 激活场景 |
|------|------|---------|
| 出题模式 | `contexts/出题模式.md` | 出题命令 |
| 批改模式 | `contexts/批改模式.md` | 批改命令 |
| 讲解模式 | `contexts/讲解模式.md` | 语法答疑、学习记录 |
| 复习模式 | `contexts/复习模式.md` | 复习计划、薄弱点查看 |

### 持续学习系统

| 文件 | 用途 |
|------|------|
| `continuous-learning-config.md` | 错误模式提取规则、模式生命周期、出题联动 |
| `错误模式库.md` | 实际存储的模式数据 |

## 执行原则

1. **收到触发词 → 读 command 文件 → 读 context 文件 → 按步骤执行**。不凭记忆，每次都读文件。
2. **需要某个规则时 → 读对应的 `rules/` 文件**。CLAUDE.md 只是索引，具体规则在独立文件中。
3. **批改是流水线** — 6 个阶段按顺序执行，每阶段的产出是下一阶段的输入。
4. **持续学习** — 每次批改都自动提取/更新错误模式，出题时参考模式库针对性出题。
5. **Git 自动管理** — 规则见 `rules/git工作流.md`。
