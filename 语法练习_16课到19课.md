## 一、动词变形（共15题）

把下列动词变成指定的形式。

### A. 变成「ない形」

1. 書きます → 書かない
2. 飲みます → 飲まない
3. 帰ります → 帰らない
4. 食べます → 食べない
5. 来ます → 来ない
6. します → しない
7. 買います → 買わない
8. 泳ぎます → 泳がない
9. 話します → 話さない
10. 使います → 使わない

### B. 变成「原形（辞書形）」

11. 働きます → 働く
12. 教えます → 教える
13. 勉強します → 勉強する
14. 寝ます → 寝る
15. 乗ります → 乗る

## 二、形容词/名词て形变形（共8题）

把下列形容词或名词变成「て形」，连接后面的句子。

例：安い／おいしい → 安くて、おいしいです。

1. 高い／大きい → 高くて、大きいです。
2. 新しい／便利 → 新しくて、便利です。
3. 静か／綺麗 → 静かで、綺麗です。
4. 賑やか／楽しい → 賑やかで、楽しいです。
5. 元気／親切 → 元気で、親切です。
6. 学生／20歳 → 学生で、20歳です。
7. 暑い／大変 → 暑くて、大変です。
8. 有名／古い → 有名で、古いです。

## 三、填助词（共10题）

从「は・が・を・に・で・へ・と・から・まで・も」中选择合适的助词填入括号中。每个括号填一个。

1. 毎朝7時（に）起きます。
2. 私は東京（に）住んでいます。
3. 次の駅（で）電車（を）降ります。
4. 日本語（は）できますか。【错误：应该是が】
5. 金曜日（までに）（は）レポートを出してください。
6. あの人（は）髪（が）長いです。
7. 日本（で）一番高い山は富士山です。

## 四、选择正确答案（共12题）

从括号里选一个正确的答案，写在 → 后面。

1. ここで写真を（撮って／撮らないで）ください。禁止です。
→ 撮らないで

2. パスポートを（忘れても／忘れないで）ください。
→ 忘れないで

3. 明日は日曜日ですから、早く起き（なければなりません／なくてもいいです）。
→ なくてもいいです

4. 毎日薬を飲ま（なければなりません／なくてもいいです）。医者に言われました。
→ なければなりません

5. この漢字の読み方を知って（います／あります）か。
→ います

6. 日本語の新聞を読む（ことができます／ことができません）。難しいですから。
→ ことができません

7. 寝る（前に／後で）、歯を磨きます。
→ 後で【错误：应该是前に】

8. ミラーさんは（もう／まだ）帰りましたか。——いいえ、（もう／まだ）です。
→ もう、まだ

9. この映画は（どれ／どの）映画館で見ることができますか。
→ どの

10. お茶を（もう／また）一杯いかがですか。
→ また【错误：应该是もう】

11. 日曜日（まで／までに）本を返してください。
→ までに

12. 日本語は（なかなか／ぜひ）上手になりません。
→ なかなか

## 七、造句/翻译题（共10题）

请把下列中文翻译成日语。

1. 请不要在这里抽烟。

→ ここへタバコは吸わないでください。【错误：应该是ここでたばこを吸わないでください。】

2. 明天之前必须写完报告。

→ 明日までにレポとは書かなければなりません。【错误：应该是明日までにレポートを書かなければなりません。】

3. 明天不来也行。

→ 明日来なくてもいいです。

4. 我会弹钢琴。

→ 私はピアノが弾きあります。【错误：应该是私はピアノを弾くことができます。】

5. 吃饭之前洗手。

→ ご飯たべまでに手が洗うます。【错误：应该是ご飯を食べる前に、手を洗います。】

6. 大阪很热闹，而且食物很好吃。

→ 大阪は賑やかで、食べ物がおいしいです。

7. 那位老师年轻又亲切。

→ あの先生は若くて、親切です。

8. 你知道那家餐厅吗？

→ あのレストランはありますか？【错误：应该是あのレストランを知っていますか。】

9. 渐渐变冷了。

→ 続々寒いなります。【错误：应该是だんだん寒くなりました。】

10. 我怎么也记不住汉字。

→ 私わ漢字がなかなか覚えないです。【错误：应该是なかなか漢字を覚えることができません。或 漢字がなかなか覚えられません。】

## 八、まだ・もう・また・も 辨析（共6题）

从「まだ・もう・また・も」中选择合适的词填入括号。

1. A：宿題はできましたか。　B：いいえ、（まだ）です。
2. （もう）6時ですね。帰りましょう。
3. 明日（も）来てください。【错误：应该是また】
4. 日本語（も）英語（も）できます。
5. 誰（も）いません。
6. 楽しかったです。（もう）行きたいです。【错误：应该是また】

## 九、综合读解（共1题）

阅读下面的短文，回答问题。

> 私は毎朝6時に起きます。起きてから、顔を洗います。朝ごはんの前に、30分ぐらい日本語を勉強します。朝ごはんを食べてから、8時に家を出ます。電車に乗って、会社へ行きます。会社は9時から5時までです。毎日忙しくて、大変です。でも、仕事は楽しいです。

**问题：**

1. この人は毎朝何時に起きますか。
→ 6時に起きます。

2. 朝ごはんの前に何をしますか。
→ 30分ぐらい日本語を勉強します。

3. 何で会社へ行きますか。
→ 電車に乗って、会社へ行きます。

4. 会社は何時から何時までですか。
→ 9時から5時までです。

5. 仕事はどうですか。
→ 忙しくて、大変ですが、楽しいです。

## 十、纠错题（共5题）

下列句子各有一处错误，请找出来并改正。

1. この部屋は静かくて、いいです。
→ 错误：静かくて　改正：静かで

2. ここで写真を撮らなくてください。
→ 错误：撮らなくて　改正：撮らないで【错误：改成了撮らなくで，应该是撮らないで】

3. 明日までにレポートを書きます。（提示：写报告是持续动作）
→ 错误：までに　改正：前に

4. 日本語を話すことをできます。
→ 错误：ことを　改正：ことが

5. あの人は誰も知りません。
→ 错误：は　改正：が【错误：应该是を】
